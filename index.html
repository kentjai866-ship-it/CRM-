☁ Cloud opslag — Supabase

Sla je data op in de cloud zodat je hem vanaf meerdere apparaten kunt gebruiken. Elke opslag wordt automatisch gesynchroniseerd.

Status:
○ Niet verbonden
Stap 1 — Supabase project koppelen
  1. Maak een gratis account op supabase.com
  2. Maak een nieuw project aan
  3. Ga naar Project Settings → API en kopieer de Project URL en anon public sleutel
  4. Voer de SQL hieronder uit in de SQL Editor van je project
SQL uitvoeren in Supabase → SQL Editor:
create table if not exists public.app_data (
  user_id  uuid references auth.users not null,
  bedrijf  text not null,
  entiteit text not null,
  data     jsonb not null default 'null',
  bijgewerkt_op timestamptz not null default now(),
  primary key (user_id, bedrijf, entiteit)
);
alter table public.app_data enable row level security;
create policy "Eigen data" on public.app_data
  for all using (auth.uid() = user_id)
  with check (auth.uid() = user_id);

Backup & herstel

🔄 Data overzetten naar nieuwe computer / browser
  1. Klik Download backup — sla het .json bestand op (bijv. USB of e-mail naar jezelf)
  2. Open facturatie_9.html op de nieuwe computer in je browser
  3. Ga naar Instellingen → Backup & herstel → kies het .json bestand bij Herstel van backup
✓ Backup bevat: CKL + T&B Beheer · facturen · contacten · uitgaven · uren · projecten · offertes
